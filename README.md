# 📊 Customer Churn Prediction System

## Machine Learning Web Application for Predicting Customer Retention Risk


## 📌 Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. Losing customers affects revenue, growth, and long-term customer relationships.

This project develops a **Customer Churn Prediction System** that uses Machine Learning to predict whether a customer is likely to leave a service provider.

The system analyzes customer demographic information, service usage, billing details, and subscription information to classify customers into:

- ✅ **Customer Will Stay**
- ⚠️ **Customer Will Churn**


The trained machine learning model is deployed using **Flask**, providing an interactive web application where users can enter customer details and receive instant predictions.





# 🎯 Project Objectives

The main objectives of this project are:

- Analyze customer behavior patterns affecting churn
- Perform exploratory data analysis (EDA)
- Clean and preprocess customer data
- Train machine learning classification models
- Evaluate model performance
- Deploy the final model as a web application
- Provide a user-friendly prediction interface




# 🏢 Business Problem

Customer acquisition is usually more expensive than customer retention.

Businesses need to identify customers who are likely to leave so they can take preventive actions such as:

- Offering discounts
- Providing better support
- Improving customer experience
- Creating personalized retention campaigns


This project helps businesses make data-driven decisions by identifying high-risk customers.
